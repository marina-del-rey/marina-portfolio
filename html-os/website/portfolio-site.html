    Portfolio site

    [marinavn.net]

  • An immersive app used to showcase my work. Visitors can explore an interactive 3D scene of my room, modeled in detail, which leads to a playful HTML/CSS/JavaScript recreation of the classic Windows95 operating system. Within this simulated OS, users can view my portfolio through a virtual browser, adding a touch of nostalgia to the browsing experience.

  • I developed all aspects of the app, including designing and modeling the 3D room, building the simulated OS, and setting up and configuring the server for optimal performance and reliability.

    • Stack: React, React3Fiber, Express, Node.js, Nginx, Docker

    Trivial

    [trivial.marinavn.net]

  • A trivia app where users can customize their quiz experience by selecting categories and difficulty levels, tracking their score in real-time as they race against a timer for each question. After completing a quiz, users have the option to add their score to a public leaderboard.

  • I created, designed, and configured the web app, setting up and managing the database while handling deployment, server configuration, and database administration.

    • Stack: React, Express, Node.js, SQLite, Nginx, Docker

    My Diary

    [mydiary.online]

  • A diary app that allows users to create, manage, and explore multiple diaries with detailed entries. Users can log in to create diaries, add entries and view shared diaries from other users. The application provides an intuitive interface for easy navigation and discovery, allowing users to seamlessly create, browse, and share diary entries in a connected community.

  • I designed, developed, and configured the entire application, setting up the API and managing the MongoDB database. I also configured the SMTP server for password recovery and email verification, handled deployment, and oversaw server and database administration.

    • Stack: React, Express, Node.js, Mongodb, Nginx, Docker, Postfix, Dovecot
